# @coreviz/sdk

CoreViz SDK - A JavaScript/Node.js interface for CoreViz's Vision AI APIs.

## Installation

```bash
npm install @coreviz/sdk
```

## Usage

```js
const coreviz = require('@coreviz/sdk');
coreviz();
```
